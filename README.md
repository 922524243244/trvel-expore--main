# 🌍 Wanderly – Professional Travel & Holiday Booking Platform

A modern, responsive travel and holiday booking website that helps users explore destinations, book tours, reserve hotels, discover transport services, and plan unforgettable trips.

---

## 📌 Project Overview

Wanderly is a frontend travel booking platform designed to provide users with an interactive and visually appealing travel experience. The website features destination exploration, tour packages, hotel booking, transport information, food recommendations, travel galleries, customer reviews, and user authentication.
